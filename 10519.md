# E-MTAB-10519 数据集集成计划（修订版）

## 背景

E-MTAB-10519-raw 是 ArrayExpress 公开的单细胞 RNA-seq 数据集（黑腹果蝇），包含 14,078 基因 × 527,530 细胞的**原始 UMI 计数**（稀疏 MTX 格式）。需要将其集成到现有的基因表达预测 pipeline 中。

### 关键数据特征
- **格式**: Matrix Market (.mtx + .mtx_rows + .mtx_cols)，7.2GB 稀疏矩阵
- **数据**: 原始 UMI counts（整数值），每 cell 总 UMI 不同
- **基因**: 14,078 FlyBase IDs (FBgn...)，mtx_rows 格式为 `FBgnXXXX\tFBgnXXXX`（两列相同）
- **细胞**: 527,530 个，61 个生物学样本（SAMEA 前缀）
- **稀疏度**: 7.3% non-zero
- **Promoter**: 复用 `data/processed/promoter_*.csv`（13,307 基因，已做 train/val/test 切分）
- **基因重叠**: MTX 基因与 promoter 基因重叠 11,062（78.6%）

## Step 1: process_data.py 增加 MTX 处理分支

### 决策: 直接在 process_data.py 中增加 `build_from_mtx()` 函数

不新建脚本，在 `scripts/process_data.py` 中新增一条处理流程：

```
MTX raw → AnnData(h5ad) + QC plots → 输出到 data/emtab_processed/
```

**新增函数: `build_from_mtx(mtx_dir, out_dir)`**

```python
def build_from_mtx(
    mtx_dir: Path,        # data/E-MTAB-10519-raw/
    out_dir: Path,        # data/emtab_processed/
    promoter_dir: Path,   # data/processed/  (复用已有 promoter CSV)
) -> None:
```

**处理步骤:**

1. `mmread` 读取 MTX (cell × gene CSR 矩阵)
2. 从 `.mtx_rows` 读 FlyBase IDs → `var["gene_id"]`（两列取第一列即可）
3. 从 `.mtx_cols` 读 cell barcodes → `obs_names`，解析 SAMEA 前缀 → `obs["sample_id"]`
4. **不调用 `add_gene_annotation()`** — mtx_rows 中已有 gene_id，通过 gene_id 直接与 promoter CSV 匹配
5. **线粒体基因**: 从 GTF 获取 `chr == "mitochondrion_genome"` 的 gene_id 列表，在 `var["mt"]` 中标记 True/False
6. 调用 `sc.pp.calculate_qc_metrics(qc_vars=['mt'])` 计算 total_counts, n_genes_by_counts, pct_counts_mt
7. **QC 绘图**（3 个子图，参考 `filter_cells(plot_qc=True)`）:
   - (a) `total_counts` 分布直方图 + top 10% cutoff 线
   - (b) `total_counts` vs `n_genes_by_counts` 散点图（density 着色: `c=密度, cmap='viridis'`）
   - (c) `total_counts` vs `pct_counts_mt` 散点图
8. **gene_id 验证**: 随机抽取 10 个 gene_id，输出到 `out_dir/gene_check.csv`（含 gene_id 和对应的 top expressing cell type）
9. 保存为 `integrated_data.h5ad`，含 `layers['counts']`（原始 UMI）
10. 取 top 10% total_counts 细胞 → 保存为 `integrated_data_hq.h5ad`
11. 复制 promoter CSVs 到 out_dir

**输出结构:**
```
data/emtab_processed/
├── integrated_data.h5ad          # 全部 527K 细胞
├── integrated_data_hq.h5ad       # top 10% 细胞
├── promoter_train.csv            # 从 data/processed/ 复制
├── promoter_val.csv
├── promoter_test.csv
├── qc_total_counts.png
├── qc_counts_vs_genes.png
├── qc_counts_vs_mito.png
└── gene_check.csv                # 手动验证用
```

### 线粒体基因标注（精简版）

不调用完整的 `add_gene_annotation()`，只从 GTF 提取 mito 基因:

```python
def _get_mito_genes(gtf_path: Path) -> set[str]:
    """从 GTF 获取 mitochondrion_genome 上的 gene_id 集合"""
    import pyranges as pr
    gtf = pr.read_gtf(str(gtf_path))
    df = gtf.df
    if "Feature" in df.columns:
        df = df[df["Feature"] == "gene"]
    return set(df[df["Chromosome"] == "mitochondrion_genome"]["gene_id"])
```
